بدي تبنيلي موقع متجر حلويات احترافي باسم **"lecker"** يكون جاهز للإستخدام الحقيقي (Production Ready)، ومهم جدًا يكون نظيف، قابل للتعديل، وقابل للبيع لزبائن.

⚠️ مهم جدًا:
أنا ما بدي مشروع تجريبي أو معقد — بدي مشروع بسيط، مستقر، يشتغل مباشرة بدون مشاكل.

---

## ❗ مشاكل واجهتها سابقًا (لا أريدها تتكرر نهائيًا):

* مشاكل pnpm workspace
* مشاكل optional dependencies (rollup / lightningcss)
* مشاكل Windows compatibility
* مشاكل environment variables (.env ما ينقرأ)
* مشاكل build و dev scripts (مثل export تبع Linux)
* مشاكل dependencies مثل catalog: أو workspace:*
* مشاكل Vite بسبب native modules
* مشاكل تثبيت وتشغيل معقدة

👉 المشروع لازم يكون خالي تمامًا من كل هذه المشاكل.

---

## ✅ التشغيل:

* المشروع يشتغل مباشرة بعد:

  * npm install
  * npm run dev
* بدون أي تعديل إضافي
* بدون pnpm
* بدون أي تعقيد

  * إحصائيات عامة

* /admin/orders.html → عرض:

  * كل الطلبات

* /admin/revenue-monthly.html → أرباح الشهر

* /admin/settings.html → إعدادات:

  * تشغيل/إيقاف المتجر
  * التحكم بالتوصيل
  * إعدادات عامة
