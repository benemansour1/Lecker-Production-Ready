🎯 المطلوب

1. المشروع يجب أن يعمل مباشرة بعد التحميل بدون أي أخطاء

 لا errors في console

 لا مشاكل Tailwind أو CSS

2. بيئة التشغيل

 يجب أن يعمل على Windows (PowerShell) بدون أي مشاكل

 جميع الأوامر تعمل مباشرة بدون تعديل

---

3. Firebase (إجباري)

 استخدام Firebase فقط (Firestore + Auth + Storage)
 لا أريد PostgreSQL أو Drizzle أو أي database أخرى
 المشروع يجب أن يعمل حتى لو Firestore غير مفعل (مع fallback بسيط أو mock data)
 عدم رمي errors إذا Firestore غير جاهز

---

4. البيانات (مهم جداً)

 المنتجات تكون جاهزة داخل المشروع (seed أو JSON)
 الموقع يعرض منتجات مباشرة بدون الحاجة لإعداد database
 عند ربط Firebase لاحقاً يتم التحويل بدون مشاكل

5. CSS  Tailwind

 إعداد Tailwind بشكل صحيح 100%
 إضافة

   tailwind.config.js
   postcss.config.js
 بدون أخطاء

   lightningcss
   postcss plugin errors
 الموقع يكون responsive بالكامل (mobile-ready)

---

6. Frontend

 React + Vite
 لا errors أثناء التشغيل
 لا مشاكل import أو build

---

7. Admin + User

 فصل كامل بين

   admin
   user
 روابط منفصلة
 حماية routes

8. Real-time

 الطلبات والمنتجات تحدث مباشرة (Firestore realtime)

9. Firebase setup

 إعداد Firebase config جاهز داخل المشروع
 مع شرح واضح

   أين أضع config
   كيف أفعل Firestore
 بدون ما يوقف الموقع إذا Firebase غير جاهز

10. تشغيل المشروع
    أريد أمر واحد فقط لتشغيل كل شيء

## ⚠️ المشاكل التي واجهتها سابقًا (ممنوع تتكرر):

* صفحة بيضاء عند تشغيل الموقع (blank page)
* أخطاء Firebase مثل:

  * Service firestore is not available
* وجود API server غير ضروري وتعقيد المشروع
* تعارض في node_modules ووجودها في أكثر من مكان
* أخطاء tsconfig ومراجع لمجلدات غير موجودة
* ربط API بدل Firebase (أنا لا أريد API نهائيًا)
* مشاكل في React Query (No QueryClient set)
* المنتجات لا تظهر لأن Firestore فارغ
* الحاجة لإدخال المنتجات يدويًا (هذا غير مقبول)
* أخطاء proxy و /api (لا أريد أي API)
* مشاكل في المسارات (paths / aliases)
* مشاكل Vite config
* مشاكل بسبب اختلاف النظام (أنا أستخدم Windows وليس Mac)
